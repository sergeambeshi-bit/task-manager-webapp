# Task Manager WebApp

Full-stack task management application with login, CRUD operations, API integration, and responsive UI.

## Features
- User authentication
- Task creation, editing, deletion
- Responsive user interface
- API integration

## Tech Stack
- HTML, CSS, JavaScript
- Node.js / Express
- MongoDB

## Usage
```bash
git clone https://github.com/sergeambeshi-bit/task-manager-webapp.git
cd task-manager-webapp
npm install
npm start
